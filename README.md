# shigatoxin-galaxy
This tool implemented for use in a Galaxy instance (https://galaxyproject.org/) performs Shiga toxin sub-typing of Escherichia coli

requirements for the tool:  
python 3.7  
perl 5.26.2  
perl-bioperl 1.7  
blast 2.9  
trimmomatic 0.39  
spades 3.14  
skesa 2.3  
fastqc 0.11.9  
muscle 3.8
